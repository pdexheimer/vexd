The Virus Expression Database (VExD) contains a comprehensive list of bulk RNA-Seq and microarray experiments that involve direct infection of human cells or tissue with a virus. All data has been uniformly processed using a common gene reference (see Methods) so that differentially expressed genes are as comparable as possible.

VExD currently contains {{ study_count }} different studies from GEO tracking infection of {{ virus_count }} distinct viruses.

Look up specific genes to see their regulation and p-value in all viral infection experiments. Note that some experiments, particularly those that use older microarrays, may not query all genes.

More detailed instructions can be found on the Help page.
